Installation
=============

Download the `pre-built module <https://radarsimx.com/product/radarsimpy/>`_, and put the radarsimpy folder within your project folder as shown below:

**Windows**

::

    - your_project.py
    - your_project.ipynb
    - radarsimpy
        - __init__.py
        - radarsimcpp.dll
        - simulator.xxx.pyd
        - rt.xxx.pyd
        - radar.py
        - processing.py
        - ...


**Linux**

::

    - your_project.py
    - your_project.ipynb
    - radarsimpy
        - __init__.py
        - libradarsimcpp.so
        - simulator.xxx.so
        - rt.xxx.so
        - radar.py
        - processing.py
        - ...

**MacOS**

::

    - your_project.py
    - your_project.ipynb
    - radarsimpy
        - __init__.py
        - libradarsimcpp.dylib
        - simulator.xxx.so
        - rt.xxx.so
        - radar.py
        - processing.py
        - ...
